# 同类游戏系统调研

## 一、下落式/判定类音乐游戏综述

"下落式判定"是指：音符沿固定方向（通常从上往下）移动，玩家在音符到达判定线时按键，系统根据时机偏差给出评级。

---

## 二、主要游戏系统拆解

### 1. Beatmania IIDX（街机，1999-）
**定位**：竖直下落 + 转盘，硬核核心玩法

| 维度 | 详情 |
|------|------|
| 音轨数 | 7键 + 1转盘（SP），14键 + 2转盘（DP） |
| Note类型 | 普通键、长按键（CN）、地狱长按（HN） |
| 判定等级 | PGREAT > GREAT > GOOD > BAD > POOR |
| 判定窗口 | PGREAT ±16.7ms，GREAT ±33.4ms |
| 计分 | 按 PGREAT/GREAT 数量计算，满分 MAX（理论上每个音符2分） |
| 血条 | 默认有血条（失败系），可选EASY/HARD/EX-HARD档位 |
| 连击 | 有COMBO，但不影响分数 |
| 评级 | AAA > AA > A > B > C > D > E > F |
| 特色 | 高精度判定窗，Hard模式血量极苛刻 |

### 2. Sound Voltex（SDVX）（街机，2012-）
**定位**：4键+2FX键+2模拟旋钮，视觉张力强

| 维度 | 详情 |
|------|------|
| 音轨数 | 4BT + 2FX + 2旋钮（模拟） |
| Note类型 | BT单键、BT长按、FX单键、FX长按、旋钮（连续曲线） |
| 判定 | S-CRITICAL > CRITICAL > NEAR > ERROR |
| 计分 | 满分10,000,000，CRITICAL=2分，NEAR=1分 |
| 血条 | 错误扣血，降至0清版；可选BLASTIVE/PERMISSIVE |
| 旋钮 | 跟随曲线转动旋钮，偏差过大算ERROR |
| 特色 | 旋钮机制带来独特触感，视觉特效与音符高度结合 |

### 3. Chunithm（街机，2015-）
**定位**：滑条+空中动作，无需触摸

| 维度 | 详情 |

| 音轨数 | 16格宽（类似钢琴）连续位置 |

| 判定 | JUSTICE CRITICAL > JUSTICE > ATTACK > MISS |

| 评级 | SSS+ > SSS > SS > S > AA > A > BBB...C |

### 4. maimai（街机，2012-）
**定位**：圆形布局，8方向

| 维度 | 详情 |
|------|------|
| 布局 | 圆形屏幕，8个TAP键位 + 全屏触摸区 |
| Note类型 | TAP、HOLD、SLIDE（带箭头轨迹）、TOUCH（屏幕任意位置）、TOUCH HOLD、BREAK |

| 判定 | CRITICAL PERFECT > PERFECT > GREAT > GOOD > MISS |

| 计分 | 理论满分 101%（含BREAK加分） |
| 评级 | SSS+ > SSS > SS+ > SS > S+ > S > AA > A...D |

## 四、传统下落式核心要素提炼

以下是"传统下落式/判定类"游戏的**共性核心要素**：

### 4.1 音符系统
- **单击（Tap/Click）**：最基础，瞬间触发
- **长按（Hold/Long Note）**：按住保持，有头部和尾部判定
- **滑动（Slide/Flick）**：在特定方向滑过

### 4.2 判定系统
- 通常分 4~6 档（PERFECT / GREAT / GOOD / BAD / MISS）
- 精度越高的档次给分越多，MISS 通常给 0 分
- 判定窗口（ms）是核心调参项

### 4.3 计分系统
- 基于判定档位 × 权重
- COMBO 倍率 或 固定分制（100万/1000万等整数满分）
- 准确率（Accuracy）单独统计

### 4.4 连击（COMBO）
- 不 MISS 则持续增加
- 影响结算评价，不一定直接影响分数

### 4.5 关卡结算
- 数值评级：分数、准确率
- 字母评级：基于分数或准确率区间（S/A/B/C/D）
- 特殊评级：全连（FC）、全完美（AP/P）

### 4.6 谱面系统
- BPM（速度）与小节/拍位绑定
- 音符位于时间轴上的精确位置
- 滚速（落下速度）可独立于BPM调节
