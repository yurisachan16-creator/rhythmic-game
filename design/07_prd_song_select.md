**版本：** v1.0
**日期：** 2026-03-18
**状态：** 待评审

---

## 一、目标与范围

### 1.1 功能目标
玩家进入选曲界面后，能够**浏览曲目列表、选择难度、查看个人记录**，并最终确认进入游玩。

### 1.2 不在范围内（本版本）
- 搜索 / 过滤 / 排序（预留位置，后续迭代）
- UGC 谱面导入入口（单独功能）
- 排行榜查看（后续联网功能）

## 四、曲目列表区

### 4.1 卡片设计

每首曲目显示为一张**卡片**，列表垂直排列：

| 状态 | 卡片样式 |
|------|---------|
| **选中（焦点）** | 全尺寸，高亮边框，封面+标题+艺术家+当前难度等级标签 |
| **相邻（非焦点）** | 缩小至 80%，透明度 60%，仅显示标题 |
| **远端（不可见）** | 透明度 0，不渲染 |

**选中卡片内容：**
```
┌──────────────────────────────────┐
│ [封面缩略图 64×64]  歌曲名称      │
│                    艺术家        │
│                    [HARD Lv.8]  │
│                    ■■■■■□□ 8/15 │  ← 难度可视化
└──────────────────────────────────┘
```

### 4.2 可见卡片数量
- 屏幕内同时显示：选中卡片 1 张 + 上方最多 3 张 + 下方最多 3 张
- 总计最多同时渲染 7 张卡片
- 超出范围的卡片隐藏（`visible = false`），不销毁

### 4.3 列表循环
- 列表**不循环**（到顶/到底停止）
- 到达边界时，对应方向的导航操作无效果（卡片轻微"弹动"提示）

## 五、曲目详情区

### 5.1 封面图
- 尺寸：256×256（固定）
- 切换曲目时：旧封面淡出，新封面淡入（动画时长 150ms）
- 无封面时：显示默认音符图标

### 5.2 基本信息
| 字段 | 来源 | 说明 |
|------|------|------|
| 歌曲名 | meta.json → title | 若有 title_unicode 则优先显示 |
| 艺术家 | meta.json → artist | 同上 |
| BPM | meta.json → bpm | 若为变BPM，显示"120-180" |
| 时长 | 音频文件解析 | 格式 m:ss |

### 5.3 难度选项卡
- 以 Tab 形式横排，数量由该歌曲的谱面数决定（1~4个）
- 每个 Tab 显示：难度标签（EASY/NORMAL/HARD/EX）+ 数字等级
- **选中的 Tab 高亮**，对应详情区内容联动更新
- 若某难度未解锁（后续扩展），Tab 显示锁定图标，不可选

### 5.4 最佳记录展示

**有记录时：**
```
最佳成绩
[评级图标]  分数: 987,654
            Accuracy: 97.43%
            最高连击: 856
            [FC] [新纪录]       ← 徽章，按实际获得情况显示
```

**无记录时（该难度未游玩）：**
```
最佳成绩
[ - ]  尚无记录
```

**徽章优先级（互斥显示最高）：**
- `AP`（All Perfect）> `FC`（Full Combo）> `GFC` > 无徽章

### 5.5 开始按钮
- 始终可见，位于详情区底部
- 显示当前确认键提示：`▶  开始游玩  [SPACE / 左键]`
- 按钮悬停时高亮（鼠标操作提示）

## 六、交互设计

### 6.1 输入映射

| 操作 | 键盘 | 鼠标 |
|------|------|------|
| 向下（下一首） | ↓ 或 S | 滚轮向下 |
| 向上（上一首） | ↑ 或 W | 滚轮向上 |
| 切换到下一难度 | → 或 D | — |
| 切换到上一难度 | ← 或 A | — |
| 确认进入游玩 | Space 或 Enter | 左键单击开始按钮 |
| 返回主菜单 | ESC | 左键单击返回按钮 |
| 快速滚动 | 长按 ↑/↓（连发） | — |

> 说明：鼠标左键点击曲目列表中的**非选中卡片**，效果等同于导航到该卡片（不直接确认）。
> 需要再次点击（或点击开始按钮/按空格）才能确认进入。

### 6.2 导航行为细节

**单次导航（按一次下）：**
1. 列表动画滚动到下一首（动画时长 120ms，缓出曲线）
2. 右侧详情区更新（封面淡入/淡出，150ms）
3. 预览音乐：等待 **800ms**（防止快速滑过时频繁切歌）后，开始播放预览片段

**连续快速导航（长按或快速滚轮）：**
1. 列表快速滚动，动画时长缩短至 60ms
2. 详情区暂停更新（避免频繁刷新）
3. 停止操作后 300ms，更新详情区和预览音乐

**切换难度：**
1. 对应 Tab 高亮切换（动画 80ms）
2. 最佳记录区联动刷新
3. 不重新触发预览音乐（预览音乐不变）

### 6.3 预览音乐
- 预览片段：从 meta.json → preview_start 指定的秒数开始，播放 **15 秒**后淡出循环
- 音量：BGM 音量的 80%（略低于游玩时）
- 切曲时：前一首 **300ms 淡出** → 新曲 **300ms 淡入**
- 进入游玩时：立即停止预览（无淡出，游玩音乐接管）

### 6.4 确认进入流程
```
玩家按 Space / 左键点击"开始游玩"
  → 开始按钮播放点击动画（100ms 压缩效果）
  → 界面整体淡出（300ms）
  → 传递参数给 GameManager：{ song_meta, difficulty }
  → GameManager 切换到 GameplayScene
```

### 6.5 返回流程
```
玩家按 ESC / 左键点击"← 返回"
  → 预览音乐淡出（300ms）
  → 界面整体淡出（200ms）
  → 切换到 MainMenu
```

## 七、状态定义

```
SongSelectState:
  ENTERING        # 进场动画播放中，输入锁定
  BROWSING        # 常态，可自由导航
  NAVIGATING      # 导航动画进行中（120ms），输入缓冲（不丢弃）
  CONFIRMING      # 确认动画/淡出进行中，输入锁定
  EXITING         # 退出动画进行中，输入锁定
```

**输入缓冲说明（NAVIGATING 状态）：**
- 动画播放期间收到的导航输入**缓冲保留 1 个**（最新的）
- 动画结束后立即处理缓冲输入，实现连续快速切歌的流畅感

## 八、数据需求

### 8.1 界面初始化时需要的数据

| 数据 | 来源 |
|------|------|
| 全部 SongMeta 列表 | 扫描 `res://songs/` + 用户导入目录 |
| 上次选中的歌曲ID + 难度 | SaveManager（本地存档） |
| 每首歌每个难度的最佳记录 | SaveManager |

### 8.2 每次切换曲目时刷新的数据

| 数据 | 来源 |
|------|------|
| 封面图 | 异步加载（避免卡帧） |
| 各难度最佳记录 | SaveManager（内存缓存，即时） |
| BPM / 时长 | SongMeta（已在内存中） |

### 8.3 进入游玩时传递的数据

```gdscript
{
    "song_meta": SongMeta,
    "difficulty": "HARD",       # 当前选中难度字符串
    "chart_file": "charts/hard.json",  # 对应谱面文件路径
    "fail_mode": SettingsManager.get("default_fail_mode")
}
```

## 九、边界情况

| 场景 | 处理方式 |
|------|---------|
| 曲目列表为空（无歌曲） | 显示"暂无曲目，请导入谱面"提示，隐藏详情区 |
| 当前选中歌曲只有1个难度 | 难度选项卡只显示1个，左右切换无效（无提示） |
| 某难度谱面文件损坏/缺失 | 对应 Tab 显示感叹号图标，点击时提示"谱面文件异常" |
| 封面图加载失败 | 显示默认封面图，不报错 |
| 快速连续滚轮（超快）| 按"连续快速导航"规则处理，不跳帧 |
| 音频预览加载失败 | 静默跳过，无报错提示 |
| 上次存档的歌曲已被删除 | 默认选中列表第一首 |

## 十、动画规格汇总

| 动画 | 时长 | 曲线 |
|------|------|------|
| 单次导航滚动 | 120ms | 缓出（Ease Out） |
| 快速连续导航滚动 | 60ms | 线性 |
| 封面切换（淡入淡出） | 150ms | 线性 |
| 难度 Tab 切换高亮 | 80ms | 线性 |
| 预览音乐淡出 | 300ms | 线性 |
| 预览音乐淡入 | 300ms | 线性 |
| 确认按钮点击压缩 | 100ms | 缓入缓出 |
| 界面进场 | 300ms | 缓出 |
| 界面退场（确认） | 300ms | 缓入 |
| 界面退场（返回） | 200ms | 缓入 |
| 边界弹动提示 | 200ms | 弹性 |
